# Claude 交叉学科/新工科子方向独立审核

日期：2026-05-12

## 审核输入

- 对象：交叉学科/新工科，面向 2027-2031 年中国大陆本科/硕士入口岗位。
- 初始分布：6 高、8 中高、5 中、0 低、7 机会。
- 证据锚点：教育部 2026 本科专业目录及新闻稿、工信部人形机器人文件、工信部等七部门脑机接口实施意见、工信部智能制造典型场景、工业互联网行动计划、Anthropic observed exposure、AI Index、METR。

## Claude 必须修改意见

1. 可信 AI / AI 安全 / 评测治理不应保持“中”。本硕入口大量是评测报告、合规文本、基准测试和政策对齐，且与 CS 章节重叠；只有医械安全、工控安全、车规安全、红队实战和责任闭环才有保护。已采纳：上调为“中高”，并重写核心判断。
2. 侵入式 BCI / 植入器械 / 神经调控平台的机会方向入口风险不应为“中高”。本硕入口多是测试辅助、动物实验辅助、数据管理、注册资料和演示系统，核心研发更偏博士/医学/临床平台。已采纳：entryRisk 上调为“高”。
3. 空间智能 / 世界模型 / 机器人基础模型与 CS 章节重叠。已采纳：改写为“空间智能 × 建筑 / 医疗 / 地质 / 机器人场景”，强调交叉学科含义来自具体场景、传感、验证和责任链。
4. 初始 0 个低风险方向需要正面处理。已采纳：补入“机器人 / 智能装备现场联调 / 验收交付”和“医疗设备 / 有源器械硬件测试与认证”两个低直接替代方向。

## 其他采纳

- AI for Science 公共数据建模 / 自动筛选从“中高”上调为“高”，因为公共数据、候选筛选、论文复现和脚本化建模高度同质化；真正机会已经由“自驱实验室 / 材料生物闭环平台”承接。
- 整体标签从 opportunity 改为 high_variance。页面表达为“高波动”：政策和学术上行真实，但本硕入口高度分化。
